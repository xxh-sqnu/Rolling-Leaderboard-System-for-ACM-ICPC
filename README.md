# ACM-ICPC 滚榜系统

支持100支队伍的ACM-ICPC竞赛计分与闭幕式滚榜（揭榜）系统。

## 快速开始

```bash
npm install
npm start
```

浏览器访问：
- **首页导航**: http://localhost:3000
- **管理面板**: http://localhost:3000/admin （添加队伍/题目/提交，控制比赛流程）
- **实时榜单**: http://localhost:3000/scoreboard （大屏投影实时排名）
- **滚榜仪式**: http://localhost:3000/ceremony （闭幕式揭榜动画）

## 使用流程

### 1. 准备数据
打开管理面板，点击 **"生成演示数据"** 一键创建100支队伍、12道题目和大量提交记录。

或者手动添加：队伍管理 → 题目管理 → 提交管理。

### 2. 比赛流程
管理面板中按顺序操作：
1. **开始比赛** → 计时开始，榜单实时更新
2. **封榜**（比赛最后1小时）→ 榜单冻结，新提交显示为 "?"
3. **结束比赛** → 比赛结束
4. **开始滚榜仪式** → 生成揭榜计划

### 3. 滚榜仪式
打开滚榜仪式页面，逐条揭晓封榜期间的提交：

| 操作 | 快捷键 |
|------|--------|
| 下一步 | Space / → |
| 上一步 | ← |
| 自动播放/暂停 | P |
| 速度 1x/2x/3x/5x | 1 / 2 / 3 / 5 |
| 跳到开头/结尾 | Home / End |
| 全屏 | F |

## 排名规则

按 ICPC 标准规则：
1. **解题数** 降序
2. **罚时** 升序（每次AC时间 + 20分钟 × AC前的非CE错误次数）
3. 平局打破：最后解出时间早的队伍排前面

## 管理员密码

默认密码: `admin123`

在URL后加 `?password=admin123` 或在 config.js 中修改 `adminPassword`。

## 数据存储

所有数据存储在 `data/` 目录的 JSON 文件中：
- `teams.json` — 队伍信息
- `problems.json` — 题目信息
- `submissions.json` — 提交记录
- `contest.json` — 比赛状态
- `ceremony.json` — 滚榜状态

管理面板支持数据导出/导入功能。

## 技术栈

- Node.js + Express + Socket.IO
- 原生 JS + CSS3 动画
- JSON 文件存储（零数据库依赖）
